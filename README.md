# FinalFantasy12_BattleClone

## Controls

![Optional Text](gamepad.png)

In game mode
1. LSB - Move the player
2. RSB - Move the camera
3. X - Activate HUD Mode

In hud mode
1. X - Return
2. B - Confirm
3. LSB - Move the cursor
